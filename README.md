# autocad-dwf-exporter

AutoCAD에서 FORM 레이어의 블록참조를 자동 감지하여 각 영역을 개별 DWF 파일로 일괄 내보내는 AutoLISP 스크립트입니다.

## 기능

- **자동 블록 감지**: "FORM" 레이어에 삽입된 도면 테두리 블록을 자동으로 탐색
- **다단계 탐색**: 레이어 기반 → 블록명 기반 → 수동 선택 순으로 폴백
- **스마트 정렬**: 위→아래, 좌→우 순서로 자동 정렬하여 도면 번호 부여
- **일괄 DWF 출력**: 각 블록 영역을 `도면1.dwf`, `도면2.dwf`, ... 형태로 개별 플롯
- **최소 크기 필터링**: 너무 작은 블록은 자동 제외 (1000 단위 이상만 포함)

## 전제조건

- AutoCAD (AutoLISP / Visual LISP 지원 버전)
- 테두리 블록이 **"FORM" 레이어**에 삽입되어 있어야 함
- **DWF6 ePlot.pc3** 드라이버가 설치되어 있어야 함
- CTB 파일: `samjung-2.ctb` (자동 적용)

## 사용법

### 1. 스크립트 로드

AutoCAD 명령창에서:

```
APPLOAD
```

`export_dwf_by_block.lsp` 파일을 선택하여 로드합니다.

### 2. 명령 실행

```
EXPORT-DWF
```

### 3. 저장 폴더 지정

- DWF 파일을 저장할 폴더 경로를 입력합니다.
- Enter를 누르면 현재 도면 파일이 있는 폴더에 저장됩니다.
- 폴더가 없으면 자동 생성됩니다.

## 동작 순서

1. **DWF 저장 폴더 입력** — 경로 미입력 시 현재 도면 폴더 사용
2. **FORM 레이어 블록참조 수집** — 자동 탐색 실패 시 수동 선택 가능
3. **BoundingBox 수집** — 각 블록의 영역 좌표 취득
4. **정렬** — 위→아래, 좌→우 순서로 도면 번호 결정
5. **DWF 플롯** — 각 영역을 개별 DWF 파일로 출력

## 탐색 우선순위

| 순서 | 방법 | 조건 |
|------|------|------|
| 1 | 레이어 기반 | `FORM` 레이어의 `INSERT` 객체 |
| 2 | 블록명 기반 | `삼정환경 도면 폼*` 블록명 패턴 |
| 3 | 수동 선택 | 사용자가 직접 블록 선택 |
