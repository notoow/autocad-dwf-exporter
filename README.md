# autocad-dwf-exporter

AutoCAD 도면에서 테두리 영역을 자동 감지하여 개별 DWF 파일로 일괄 내보내는 AutoLISP 스크립트 모음입니다.

## 스크립트 목록

| 스크립트 | 명령어 | 감지 방식 |
|----------|--------|-----------|
| `export_dwf_by_block.lsp` | `EXPORT-DWF` | FORM 레이어 블록참조 기반 |
| `export_dwf_by_border.lsp` | `EXPORT-DWF-BORDERS` | 색상/샘플 기반 테두리 감지 (UI 지원) |

---

## 1. export_dwf_by_block.lsp

FORM 레이어에 삽입된 블록참조를 자동 감지하여 각 영역을 DWF로 내보냅니다.

### 전제조건

- 테두리 블록이 **"FORM" 레이어**에 삽입되어 있어야 함
- **DWF6 ePlot.pc3** 드라이버 설치

### 사용법

```
APPLOAD → export_dwf_by_block.lsp 로드
EXPORT-DWF
```

### 탐색 우선순위

| 순서 | 방법 | 조건 |
|------|------|------|
| 1 | 레이어 기반 | `FORM` 레이어의 `INSERT` 객체 |
| 2 | 블록명 기반 | 블록명 패턴 매칭 |
| 3 | 수동 선택 | 사용자가 직접 블록 선택 |

---

## 2. export_dwf_by_border.lsp (UI 버전)

**다이얼로그 UI**를 통해 테두리 감지 방식을 선택하고 DWF를 일괄 내보냅니다.

### 감지 방식

#### 🎯 샘플 선택 모드
- 테두리 하나를 **클릭**하면 색상/레이어를 자동 추출
- 동일한 속성을 가진 나머지 테두리를 자동 탐색

#### 🎨 색상 입력 모드
- **ACI 색상 번호** (1=빨강, 2=노랑, 3=초록 ...)
- **RGB 값** 직접 입력 (예: R:255 G:0 B:0)

### UI 기능

- 📂 **폴더 찾아보기** — Windows 폴더 선택 다이얼로그
- 🔍 **미리보기** — 내보내기 전 감지된 테두리 개수 확인
- ⚙️ **필터 옵션** — 닫힌 폴리라인만 / 같은 레이어만 / 최소 크기
- ✏️ **출력 설정** — 파일명 접두사, 플로터 변경 가능
- 🔄 **텍스트 모드 폴백** — DCL 파일 없어도 명령줄 모드로 실행 가능

### 전제조건

- 테두리가 **LWPOLYLINE** (폴리라인/사각형)으로 그려져 있어야 함
- **DWF6 ePlot.pc3** 드라이버 설치
- `export_dwf_border_ui.dcl` 파일과 같은 폴더에 배치 (UI 사용 시)

### 사용법

```
APPLOAD → export_dwf_by_border.lsp 로드
EXPORT-DWF-BORDERS
```

---

## 공통 기능

- **스마트 정렬**: 위→아래, 좌→우 순서로 자동 정렬하여 도면 번호 부여
- **일괄 DWF 출력**: `도면1.dwf`, `도면2.dwf`, ... 형태로 개별 플롯
- **자동 폴더 생성**: 지정한 폴더가 없으면 자동 생성
- **크기 필터링**: 너무 작은 객체는 자동 제외
